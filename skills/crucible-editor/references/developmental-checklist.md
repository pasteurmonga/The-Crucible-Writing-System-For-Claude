# Developmental Editing Checklist

## Story Structure

### Opening (First 10%)
- [ ] Hook engages reader within first paragraph
- [ ] Protagonist introduced with clear want/need
- [ ] World established without info-dumping
- [ ] Tone and genre expectations set
- [ ] Inciting incident arrives at appropriate point

### Middle (10-75%)
- [ ] Rising action with escalating stakes
- [ ] Each chapter advances plot or character
- [ ] Subplots weave without overwhelming
- [ ] Antagonist presence/threat maintained
- [ ] Midpoint shift changes protagonist's approach
- [ ] All-is-lost moment lands with impact

### Ending (75-100%)
- [ ] Climax delivers on promise of premise
- [ ] All major threads resolved
- [ ] Emotional payoff earned
- [ ] Theme crystallizes through action
- [ ] Resolution provides appropriate closure

## Character Arcs

### Protagonist
- [ ] Clear want (external goal)
- [ ] Clear need (internal growth)
- [ ] Flaw that creates obstacles
- [ ] Growth arc across story
- [ ] Agency in climax resolution

### Supporting Characters
- [ ] Each serves story purpose
- [ ] Distinct voices and traits
- [ ] Own motivations (not just protagonist-helpers)
- [ ] Appropriate page time for importance

### Antagonist
- [ ] Compelling motivation
- [ ] Threat level appropriate
- [ ] Active opposition (not passive obstacle)
- [ ] Thematically connected to protagonist

## Pacing

### Scene Level
- [ ] Each scene has clear goal/conflict/outcome
- [ ] Entry point engages immediately
- [ ] Exit point creates forward momentum
- [ ] Length matches importance

### Chapter Level
- [ ] Chapters end with hooks or questions
- [ ] Variety in tension levels
- [ ] Rest beats between high-intensity sections
- [ ] No chapter feels like filler

### Overall
- [ ] First act moves quickly (25% max)
- [ ] Second act maintains momentum
- [ ] Third act accelerates to climax
- [ ] No sagging middle

## Plot Coherence

### Causality
- [ ] Events follow logically from previous
- [ ] Character decisions drive plot (not coincidence)
- [ ] Consequences match actions
- [ ] No deus ex machina

### Consistency
- [ ] World rules followed
- [ ] Character behavior matches established traits
- [ ] Timeline makes sense
- [ ] Physical logistics work

### Foreshadowing
- [ ] Major reveals properly set up
- [ ] Plants not too obvious
- [ ] All planted elements pay off
- [ ] No unfired Chekhov's guns

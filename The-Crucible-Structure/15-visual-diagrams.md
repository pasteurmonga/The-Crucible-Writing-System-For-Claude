## The Forging Metaphor

```
THE FORGING METAPHOR - STORY AS METALWORK
================================================================================

    RAW ORE                    THE FORGE                    TEMPERED BLADE
    (Beginning)                (Process)                    (End)
        |                          |                            |
        v                          v                            v
    +--------+                +----------+                  +--------+

    |        |                |          |                  |        |
    +--------+                +----------+                  +--------+

    +------------------------------------------------------------------+
    |  The ore is placed in the forge. The heat begins.                |

    +------------------------------------------------------------------+
                                    |
                                    v
    MOVEMENT II: FIRST TEMPERING
    +------------------------------------------------------------------+
    |  The first hammer blows. Shape begins to emerge.                 |
    |  The metal is worked but not yet strong.                         |
    +------------------------------------------------------------------+
                                    |
                                    v
    MOVEMENT III: THE SCATTERING
    +------------------------------------------------------------------+
    |  The metal is heated again, stretched, expanded.                 |
    |  It becomes thinner, harder, more brittle.                       |
    +------------------------------------------------------------------+
                                    |
                                    v
    MOVEMENT IV: BRIGHTEST BURNING
    +------------------------------------------------------------------+
    |  The hottest fire. The metal glows white.                        |
    |  It can take any shape now -- or shatter.                        |
    +------------------------------------------------------------------+
                                    |
                                    v
    MOVEMENT V: FINAL FORGING
    +------------------------------------------------------------------+
    |  The decisive blows. The quenching.                              |
    |  The metal becomes what it will be forever.                      |
    +------------------------------------------------------------------+

    CODA: TEMPERED BLADE
    +------------------------------------------------------------------+
    |  The blade is complete. Its purpose is revealed.                 |
    |  It is not the ore it was. It is something new.                  |
    +------------------------------------------------------------------+
```
